# Image Folder Map

The site keeps images in topic folders so editors can find files quickly.

- `hero/`: homepage hero artwork. Use WebP here for speed.
- `branding/`: project logo and brand marks.
- `logos/official/`: official project, EU and agency logos.
- `logos/partners/`: partner school logos.
- `gallery/`: photos used by gallery cards.
- `pwa/`: install icons. The shared 32px favicon is `images/favicon.png`.
- `uploads/`: files uploaded from `/admin/`.

Keep one canonical copy of each asset in these folders. Update references before moving or renaming a published file.
